# tic-tac-toe

A simple tic-tac-toe running on the CLI.

![tic-tac-toe](https://github.com/user-attachments/assets/5eaf10c6-9a81-4e8b-9e24-fa20d992bc6d)

## Usage

Just run the following command.

```
$ tic-tac-toe
```

## Project setup

- run app
```
$ make run
```

- test
```
$ make test
```

- build
```
$ make build
```
